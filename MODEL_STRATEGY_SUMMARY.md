## 🚀 立即行动步骤

### 步骤1：获取API Keys（5分钟）

1. **GLM-5.1**（必须）
   - 访问：https://open.bigmodel.cn/
   - 注册 → API Keys → 创建 → 复制

2. **Gemini 3.1 Pro**（强烈建议）
   - 访问：https://aistudio.google.com/
   - 获取API Key → 启用Gemini API

### 步骤2：在TRAE中添加模型（2分钟）

1. 打开TRAE → ⚙️设置 → 模型
2. 点击「+ 添加模型」
3. 选择服务商，填入API Key和模型ID
4. 测试连接

### 步骤3：开始使用（立即）

1. 阅读 `HYBRID_MODEL_STRATEGY.md` 了解任务分配
2. 选择第一个任务测试
3. 对比代码质量
