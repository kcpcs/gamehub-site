## 📞 七、回退方案

如果确实需要使用 Claude 模型：

1. 在 `.env.local` 中配置：
   ```
   ANTHROPIC_API_KEY=your_key_here
   # 或者
   FORCE_INTERNAL_MODELS=false
   ```

2. 重启开发服务器
3. 系统将自动切换回 Claude 模式
