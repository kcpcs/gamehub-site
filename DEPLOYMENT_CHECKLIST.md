---

## 🚀 快速启动指南

### 1. 本地开发
```bash
# 安装依赖
npm install

# 初始化数据库
npm run db:push
npm run db:seed

# 启动开发服务器
npm run dev
```

### 2. 生产构建
```bash
# 构建项目
npm run build

# 本地测试生产构建
npm start
```

### 3. 部署步骤
1. 注册所需服务账号
2. 配置环境变量
3. 连接数据库
4. 部署到平台
5. 测试生产环境
