## 📊 当前问题分析

### 图片质量问题
| 问题 | 描述 | 影响 |
|-----|------|------|
| 图片模糊 | 从IGDB获取的图片分辨率低 | 用户体验差 |
| 图片不匹配 | 图片与游戏/攻略内容不符 | 误导用户 |
| 同质化严重 | 多个卡片显示相同图片 | 视觉单调 |
| 单图展示 | 缺乏多图轮播、动态效果 | 缺乏吸引力 |

### 当前图片来源
- IGDB API - 提供游戏封面图
- picsum.photos - 占位符图片（质量不可控）
- 数据库存储的cover_url字段
