## 1. 搜索引擎提交

### 1.1 Google Search Console

#### 步骤：
1. 访问 [Google Search Console](https://search.google.com/search-console/)
2. 添加新网站（使用域名验证方式）
3. 验证所有权：
   - DNS 记录验证（推荐）
   - HTML 文件验证
   - HTML 标签验证

#### 提交 Sitemap：
网站已自动生成 `/sitemap.xml`，提交地址：
```
https://yourdomain.com/sitemap.xml
```

#### 重要操作：
- ✅ 提交站点地图
- ✅ 设置首选域名（www / 非 www）
- ✅ 监控索引状态
- ✅ 查看搜索效果报告

### 1.2 Bing Webmaster Tools

#### 步骤：
1. 访问 [Bing Webmaster Tools](https://www.bing.com/webmasters/)
2. 添加网站
3. 验证所有权
4. 提交 sitemap

### 1.3 Yandex Webmaster

#### 步骤：
1. 访问 [Yandex Webmaster](https://webmaster.yandex.com/)
2. 添加网站
3. 验证所有权

## 2. 社交平台推广

### 2.1 Discord

#### 推荐操作：
1. 创建官方 Discord 服务器
2. 设置自动通知频道
3. 配置机器人自动发布新内容

#### 频道建议：
- `#announcements` - 游戏更新公告
- `#codes` - 兑换码分享
- `#guides` - 攻略讨论
- `#general` - 综合讨论

### 2.2 Twitter (X)

#### 推广策略：
1. 定期发布游戏资讯
2. 分享新兑换码
3. 互动游戏社区
4. 使用热门游戏标签

#### 推文模板：
```
🎮 【新游戏上线】《游戏名称》已加入 GameHub！
👉 查看攻略：https://gamehub.com/games/xxx
🎁 领取兑换码：CODE123

#游戏 #GameHub #游戏攻略
```

### 2.3 Reddit

#### 推广策略：
1. 在相关游戏 subreddit 发布攻略
2. 参与社区讨论
3. 避免垃圾广告

### 2.4 YouTube

#### 内容策略：
1. 游戏攻略视频
2. 兑换码合集
3. 游戏评测

## 3. 内容优化建议

### 3.1 Meta 标签优化

确保每个页面都有：
- ✅ 唯一的 `<title>`（50-60字符）
- ✅ 描述性 `<meta name="description">`（150-160字符）
- ✅ 结构化数据（Schema.org）

### 3.2 页面优化

| 要素 | 建议 |
|------|------|
| URL 结构 | 简洁、包含关键词 |
| 标题层级 | 合理使用 H1-H6 |
| 图片优化 | 添加 alt 属性 |
| 页面速度 | 启用缓存、压缩资源 |
| 移动端适配 | 响应式设计 |

### 3.3 内部链接

- ✅ 游戏页面链接到相关攻略
- ✅ 攻略页面链接到游戏
- ✅ 导航菜单清晰
